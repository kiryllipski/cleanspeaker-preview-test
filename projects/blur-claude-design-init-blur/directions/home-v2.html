Home, rethought.

Single-task launcher. Один primary action, никакого tab bar, навигация через состояние, а не destinations. Три стейта показаны рядом — first session (empty), active (есть проекты), Pro user (без апселла).

PRINCIPLE 01

One primary action — add photo

Hero «Add photo» card с двумя сегментами Camera / Library. Это главное, ради чего юзер открыл апп. Tools убраны на главную — выбор инструмента имеет смысл только когда фото уже есть.

PRINCIPLE 02

No tab bar — state, not destinations

Settings — иконкой в top-bar. Projects — через «See all» в секции Recent. Pro — inline-картой и кнопкой ✦ в top-bar. Никакой нижней навигации — освобождаем экранную площадь и убираем лишний выбор.

PRINCIPLE 03

Recent first, continue last

Если есть незавершённый проект — приоритетная карточка «Continue» над hero. Это второй по частоте паттерн использования: пришёл доделать. Дальше — горизонтальный strip recent с draft-метками.
