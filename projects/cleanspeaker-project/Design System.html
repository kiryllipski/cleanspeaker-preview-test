Figma import guide

12

html.to.design pipeline

  1. Install html.to.design plugin in Figma (free for non-commercial use).
  2. Run the plugin → Import HTML → drop this file (Design System.html).
  3. Each <section> becomes a separate Figma frame — name them as pages.
  4. Convert swatches into Color styles using Tokens Studio for Figma: paste the hex grid from §01.
  5. Convert type samples into Text styles by selecting each row in §03 and running Create style.
  6. Components (§06–10) become component sets with the demo states already on canvas.
  7. Cross-link to All Screens.html for full screen compositions, and Documentation.html for the implementation spec.
Tip — values in this file come directly from tokens.css. If you change a token there, regenerate this page and re‑import for a clean diff in Figma.
